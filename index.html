Client Description

Toolshed is a web-based company that allows users to rent tools of all kinds, from a simple hammer to a pneumatic jackhammer. It aims to democratize handyman projects by making any tool imaginable available at the click of a button. Their image is convenient, reliable, and useful.

Apparatus Design System

An apparatus is a term for the collective equipment you use for specific purposes. Toolshed aims to offer the correct tools for specific projects and offer helpful advice for the project- from big to small projects. Many may not know the name for a tool needed for a project (example: What is the name for gizmo that will tighten my squeaky chair and how do I use it?) and will need to be educated in a non-intimidating how to use it. Apparatus Design System gives an educated definition to an unknown tool that is needed to define a project.
